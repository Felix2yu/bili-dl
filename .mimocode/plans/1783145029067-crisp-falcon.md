# Plan: 替换 json-iterator/go 为 encoding/json 标准库

## 目标
去掉所有外部 JSON 依赖，使用 Go 标准库 `encoding/json`。

## 当前 jsoniter 使用分布

### api/api.go（~20 处）
- `jsoniter.Get(info, "data", "cid").ToString()` — 嵌套字段访问
- `jsoniter.Unmarshal(data, &l)` — 反序列化为 `[]map[string]any`
- `jsoniter.MarshalToString(&video)` — 序列化

### api/wbi.go（2 处）
- `jsoniter.Get([]byte(json), "data", "wbi_img", "img_url").ToString()`
- `jsoniter.Get([]byte(json), "data", "wbi_img", "sub_url").ToString()`

### api/api_test.go（1 处）
- `jsoniter.Get(bytes, "data", "cid").ToString()`

## 修改步骤

### 1. api/api.go
- 删除 `jsoniter` import，改为 `encoding/json`
- 为 bilibili API 响应定义 struct：
  ```go
  type biliVideoInfoResp struct {
      Data struct {
          Cid   string `json:"cid"`
          Title string `json:"title"`
          Page  struct {
              Count int `json:"count"`
          } `json:"page"`
      } `json:"data"`
  }
  
  type biliDashResp struct {
      Data struct {
          Dash struct {
              Video []dashStream `json:"video"`
              Audio []dashStream `json:"audio"`
          } `json:"dash"`
      } `json:"data"`
  }
  
  type dashStream struct {
      ID        float64 `json:"id"`
      Codecs    string  `json:"codecs"`
      Width     float64 `json:"width"`
      Height    float64 `json:"height"`
      Bandwidth float64 `json:"bandwidth"`
      BaseURL   string  `json:"base_url"`
  }
  
  type biliVlistResp struct {
      Data struct {
          List struct {
              VList []struct {
                  BV string `json:"bvid"`
              } `json:"vlist"`
          } `json:"list"`
      } `json:"data"`
  }
  ```
- 替换所有 `jsoniter.Get().ToString()` 为 struct 字段访问
- 替换 `jsoniter.Unmarshal` → `json.Unmarshal`
- 替换 `jsoniter.MarshalToString` → `json.Marshal` + `string()`

### 2. api/wbi.go
- 定义 `wbiImgResp` struct：
  ```go
  type wbiImgResp struct {
      Data struct {
          WbiImg struct {
              ImgURL string `json:"img_url"`
              SubURL string `json:"sub_url"`
          } `json:"wbi_img"`
      } `json:"data"`
  }
  ```
- 替换 `jsoniter.Get().ToString()` 为 struct 字段访问

### 3. api/api_test.go
- 替换 jsoniter 调用

### 4. go.mod
- `go mod tidy` 删除 json-iterator 依赖
